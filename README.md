一款多功能采集抖音评论区用户裂变截留软件，功能强大,采集流畅精准，推荐使用，支持脚本/协议自定义双模式采集抖音视频及评论区用户并导出详细信息资料。

当前软件最新8.0版本

详细功能如下(抖音视频评论区采集截留软件)：

1.支持批量导入指定视频链接采集评论区/关键词搜索视频后采集评论区/采集博主主页视频后采集评论/关键词填表采集视频后采集评论区，等多 种采集方式。

2.支持关键词直接采集用户及详细信息。

3.支持视频子评论采集评论区用户及详细信息。
